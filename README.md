# Exercícios - Programming in Haskell

---

Este repositório contém uma coletânea de resoluções de exercícios propostos na 2ª edição do livro
[**Programming in Haskell**](https://people.cs.nott.ac.uk/pszgmh/pih.html), do autor **Graham Hutton**, feitas nos meus estudos para a disciplina
eletiva de Programação Funcional no Bacharelado em Ciência da Computação na UFRJ.

O material presente aqui não substitui, de forma alguma, o livro-texto, em especial as
soluções apresentadas na obra. O compartilhamento das resoluções foi feito com propósito meramente
educativo, principalmente no que diz respeito à discussão sobre as diferentes formas de abordar os
exercícios.

Além disso, apesar de praticamente todas as respostas terem sido validadas empiricamente, não há
garantia absoluta de que todas estejam perfeitamente corretas. Sendo assim, encare as soluções daqui
com senso crítico, como objeto para comparação com soluções pensadas por conta própria. Acredito que
o conteúdo deste repositório é melhor aproveitado dessa maneira. Caso encontre qualquer problema ou
tenha alguma sugestão, sinta-se à vontade para abrir uma _issue_ ou [entrar em contato comigo](mailto:williamvqp@ic.ufrj.br).
